


## Setup & Configuration
Clone the repository:
```bash
git clone https://github.com/eskoimex/zacrac.git
```

---

## Running locally

```bash
# Install all Dev-included dependencies
npm install
# Generates Prisma cliente metadata/types stuff
npx prisma generate
```

Running the project is simple as:

```bash
npm run start
```
